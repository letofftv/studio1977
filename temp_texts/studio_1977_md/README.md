# Studio 1977: тексты сайта и личного кабинета

Комплект markdown-файлов для передачи разработчику.

## Файлы

- `01_home.md` - главная страница
- `02_about.md` - страница о студии
- `03_contact.md` - контакты
- `04_brief.md` - бриф
- `05_partner_like_it.md` - партнёрская страница направления лайк IT
- `06_dashboard_general.md` - общая логика личного кабинета
- `07_dashboard_overview.md` - дашборд
- `08_dashboard_projects.md` - проекты
- `09_dashboard_tasks.md` - задачи
- `10_dashboard_clients.md` - клиенты
- `11_dashboard_leads.md` - лиды
- `12_dashboard_team.md` - команда
- `13_dashboard_settings.md` - настройки
- `14_ui_texts.md` - общие UI-состояния, ошибки, кнопки
- `15_visual_style.md` - визуальная логика сайта и кабинета
- `16_seo_cta.md` - SEO-заголовки, описания и CTA
- `ALL_SITE_CONTENT.md` - всё в одном файле

## Общие принципы

Тон: живой, спокойный, уверенный, без агентского тумана.

Главная мысль сайта: Studio 1977 не продаёт разрозненные услуги. Студия помогает бизнесу собрать в систему маркетинг, упаковку, сайт, CRM, автоматизацию, события и коммуникацию.

Название технологического направления: `лайк IT`.

Важно: старое написание с приставкой `Ai` в этих текстах не используется.

## 2. Личный кабинет (Studio Dashboard)

Все страницы личного кабинета объединены общим дизайном с боковым меню (Sidebar).

### 2.1. Дашборд (`/studio`)
*   **Статистика**:
    *   Проектов в работе (Количество активных сделок)
    *   Новых лидов (Активные лиды за период)
    *   Задач (Всего задач пользователя)
    *   Просроченных (Задачи с истекшим дедлайном)
*   **Списки**: Вывод последних активных проектов и лидов.

### 2.2. Проекты (`/studio/projects`)
*   Таблица всех текущих сделок из Битрикс24.
*   Поля: Название проекта, Клиент, Статус, Ответственный, Дата создания.

### 2.3. Задачи (`/studio/tasks`)
*   Список персональных задач текущего пользователя.
*   Поля: ID, Описание, Крайний срок, Статус (Активна/Завершена).

### 2.4. Клиенты (`/studio/clients`)
*   База контрагентов (Компаний) из CRM.
*   Поля: Название, Тип, Оборот (регистрация доходов), Дата регистрации.

### 2.5. Лиды (`/studio/leads`)
*   Входящие заявки (Лиды).
*   Поля: Имя, Компания, Направление, Дата, Статус.

### 2.6. Команда (`/studio/team`)
*   Список сотрудников портала.
*   Поля: Имя Фамилия, Должность, E-mail, ID.

## 3. Техническая структура

*   **Технологии**: Next.js (App Router), CSS Modules, Bitrix24 REST API.
*   **Стандарт UI**: Скругление всех элементов — **4px**. Цвет акцента — золото/бронза на темном фоне.
*   **Авторизация**: OAuth 2.0 / Bitrix24 Session Cookie.
